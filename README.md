# Kiva Network Auto Mining



Automated bot for Kiva Network



## Installation



1. Clone the repository:

```bash

git clone https://github.com/raditygnacio/kiva.js.git

cd kiva.js

```



2. Install dependencies:

```bash

npm install axios

```



## Usage



Run the bot:

```bash

node kiva.js
